# AZ-500 Concept and Reference Index

> Quick-lookup index of every term, service, and acronym you will see on the exam.

- **Entra ID** - Identity
- **Conditional Access** - Identity policy engine
- **PIM** - JIT roles
- **Managed Identities** - Secret-less Azure auth
- **Azure RBAC** - Resource authorization
- **NSG/ASG** - L3/L4 ACL
- **Private Endpoint / Private Link** - Private PaaS access
- **Azure Firewall (Std/Prem)** - FWaaS
- **WAF (AppGw/Front Door)** - L7 protection
- **Bastion** - Browser RDP/SSH
- **Key Vault / Managed HSM** - Secret + key store
- **Defender for Cloud** - CSPM + CWP
- **Defender plans** - Per-workload protection
- **Microsoft Sentinel** - SIEM/SOAR
- **Azure Policy** - Compliance enforcement
