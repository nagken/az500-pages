# Common Pitfalls - AZ-500

> Frequent confusions, traps, and "gotchas" that cause wrong answers on the exam.

### Owner sprawl

Use Contributor + targeted Data roles instead of Owner everywhere.

### KV deletion risk

Always enable soft-delete + purge protection on production vaults.

### Private Endpoint DNS pitfall

PE without proper Private DNS Zone -> client resolves public IP -> firewall blocks.

### Sentinel ingestion costs

Use commitment tier + Basic/Auxiliary logs for high-volume tables.

### Defender plan sprawl

Enable per workload type with cost monitoring.
