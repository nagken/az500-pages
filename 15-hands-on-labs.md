# Hands-On Labs - AZ-500

> Free-tier or trial labs you can run end-to-end in 30-60 minutes.

### 1. Build a CA policy that requires phishing-resistant MFA for admins

Use authentication strength + role-targeted assignment.

### 2. Deploy Azure Firewall + UDR

Force a spoke's outbound 0.0.0.0/0 through firewall, allow only required FQDNs.

### 3. Add a Private Endpoint to a Storage account

Disable public network, create PE, configure Private DNS zone.

### 4. Enable Defender for Storage with malware scan

Upload an EICAR test file to verify scanning.

### 5. Author a Sentinel rule

Detect privileged role activation outside business hours; map entities; build playbook to disable user.
